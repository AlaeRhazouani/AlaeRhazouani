## Highlighted Projects

- **CI/CD Pipeline & Monitoring — Student Task Manager**
  Full CI/CD pipeline on Oracle Cloud — Jenkins Groovy DSL, Docker Compose, 3 Flask microservices, auto rollback, staging/production separation, Prometheus + Grafana observability. Everything triggered by a single `git push`.

- **AI Skills Matrix — CGI Maroc**
  AI portal analyzing software projects (ZIP + Word/PDF/Excel) to extract and map technical skills with proficiency levels and confidence scores. Built in active partnership with CGI Maroc.

- **Electric Meter OCR & QR Code**
  YOLOv8 OCR pipeline trained on a custom 2001-image dataset, detecting meter readings and QR codes in a single photo. React Native mobile app for field agents + admin dashboard + fully automated billing cycle with Dolibarr CRM sync.

- **Urban Docs Mobile App — FESURBAN**
  Cross-platform Flutter app replacing a fully paper-based process for consulting urban planning documents at the Agence Urbaine de Fès.
